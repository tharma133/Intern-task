# Anime Maze

This app is about animation which you can see how much episodes,movies are there for that animation. 

This app has two pages 1) Home page 2) Details Page

### Home Page
It contains animation images and pie chart. Pie chart show how much movies, Tv shows, etc are there in that animation. 

### Details Page
It contains title ,synopsis, episodes, ratings for the anime which you select.
